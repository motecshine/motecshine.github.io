关于未来职业一些思考

2020-02-11

Intro

总感觉自己最近很混乱，在也没有17-18年那种冲劲了，迷茫，无助，于是想理一理，自己的问题。

职场

2019年8月份从一个物联网公司离职，这家应该是我呆的最久的一家公司了，2年2个月，出来的原因也很简单，就是工资高了很多， 新的工作不是研发向的，整天CURD，久了就烦了，但是时间还是较为充裕没有996。在此期间总想做点事情，于是雨露均沾，到头来什么都没做好，这样做的后果就是什么成果都没有，也就没有了成就感，成就感对于我来说太重要了，没了它也就没了我。 我把我想要的总结下当作一个备忘录。

1. 学历

无论是内推也好还是投递也好，总给别人说自己是高中毕业，起初没觉的什么，从去年开始找工做开始四处碰壁，学历是硬伤，我准备5年(32岁)左右拿到在职研究生的学历，这个长期的计划已经在去年6月份开始实施了。

技术栈

工作这几年每家公司的技术栈都不一样，太杂了，我想了想应该需要在下面方向选择：

  • 基础技术栈

    • 计算机组成
    • 操作系统
    • 编译

  • 职业选择方向

    • Cloud Native

      这是自己现在正在做的方向， 主要是： Kubernetes Docker Mesh

    • IOT Edge

      自己一直对这个方向很感兴趣，但是无奈没有机会。

    • Distributed Database

    有意思而且TiDB Plan 有完整的学习路线。

    短期内的打算

    • 计算机组成
    • MIT 6.8.24
